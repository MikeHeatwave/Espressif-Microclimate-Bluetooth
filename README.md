# Espressif-Microclimate-Bluetooth
### 🌱 Offline-версия АСУ микроклиматом
## Компоненты системы
- 📱 Android-приложение (Kotlin)
- ⚙️ Микроконтроллер ESP32 (C++)
- 📡 Bluetooth как способ/технология связи
## Датчики микроконтроллерной подсистемы
- 🌡️ SHT31 (температура и влажность воздуха)
- 💧 Capacitive Soil Moisture Sensor (влажность почвы)
- ☀️ Фоторезистор (освещённость)
## Исполнительные устройства
- 💨 Вентилятор
- 💦 Помпа полива
- 💡 Фито-лампа освещения

## Функционал приложения для данной версии системы
1) Текущие показатели микроклимата
2) Управление устройствами
3) Настройка автоматического режима
4) Применение предустановок автоматического режима

---
## Установка
### Рекомендуется Android-устройство версии 12+
В релизах проекта скачайте ".apk" файл на устройство.

Возможно, на устройстве потребуется включить установку приложений из неизвестных источников.
